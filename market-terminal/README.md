# Aurora Market Terminal

Modern Bloomberg-inspired market workstation for GitHub Pages.

## Features

- Bundled S&P 500 constituent universe with 503 securities.
- Keyboard command bar for ticker pages and terminal functions.
- Simulated quote, chart, valuation, risk, news, portfolio, sector, heatmap, and screener views.
- Static deployment with no backend or API key.

## Commands

```text
AAPL DES
NVDA GP
MSFT FA
JPM NEWS
MARKET
MOVERS
HEATMAP
SCREEN PE<20
SECTOR TECH
PORT
HELP
```

The bundled market data is simulated for interface prototyping and should not be used for trading decisions.
